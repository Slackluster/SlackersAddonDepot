# Slacker's Addon Depot
A World of Warcraft addon. Download locations: [CurseForge](https://www.curseforge.com/wow/addons/slackers-addon-depot) | [Wago](https://addons.wago.io/addons/sad)

[![](https://media.forgecdn.net/attachments/1439/398/buymeacoffee-png.png)](https://www.buymeacoffee.com/Slackluster)

**Slacker's Addon Depot (SAD)** is a simple but powerful addon manager that lets you enable and disable addons - automatically, if you'd like.

![](https://media.forgecdn.net/attachments/1773/570/sad-01-png.png)

Create manual profiles and apply them to any character that SAD has seen before.

![](https://media.forgecdn.net/attachments/1773/571/sad-02-png.png)

Or use SAD's flagship feature: Login Profiles. This lets you enable addons automatically for matching characters. GatherMate for your herbalists and miners? Auctionator for your AH alts? Once you set them up, SAD will apply them for you automatically.

If SAD has not seen your character before, you will be prompted to reload to apply all matching Login Profiles.

> [!WARNING]
> Addons not in any matching login profiles will be disabled on login. This is because multiple login profiles can be valid for any one character. If you choose to work with Login Profiles, it is recommended to take a moment to set them up properly.

![](https://media.forgecdn.net/attachments/1773/572/sad-03-png.png)

Other features:
- A character list with all seen characters, including those from different realms.
- Use Shift+click to quickly (de)select multiple addons.

## If you are experiencing issues
- Check if the issue occurs with only this addon enabled, or if the issue is a feature you can disable from the addon's settings menu.
- If you run into any issues or if you have other feedback, feel free to join the [Discord](https://discord.gg/FwWZZ39kPX) and ask in the `#slackers-addon-depot` channel, or create a [ticket on GitHub](https://github.com/Slackluster/SlackersAddonDepot/issues). :)
